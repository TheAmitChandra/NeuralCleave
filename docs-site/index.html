Quick install

bash
# Linux / macOS
curl -fsSL https://neuralcleave.com/install.sh | bash

# Windows (PowerShell)
iwr -useb https://neuralcleave.com/install.ps1 | iex

# Or via pip directly (Python 3.12+ required)
pip install neuralcleave
neuralcleave init
neuralcleave start
💡

Python 3.12+ required. Redis and Qdrant are optional — the gateway runs without them but with reduced memory capability. See requirements.
