# neuralcleave v2 — Personal AI Assistant
#
# Single consolidated requirements file. Previously split across 7 files
# (requirements.txt, requirements-v2.txt, requirements-dev.txt,
# requirements-rag.txt, requirements-test.txt, backend/requirements.txt,
# backend/requirements-ci.txt) — most of those were leftovers from the
# pre-pivot enterprise codebase (Celery, PostgreSQL/asyncpg, Neo4j,
# LlamaIndex/RAG, OpenTelemetry, HashiCorp Vault) that v2 doesn't use at
# all. Consolidated into this one file, built from an actual scan of every
# third-party import across neuralcleave/, tests/, and scripts/ — not
# copied from the old files, which had drifted from the real codebase
# (e.g. fastapi==0.115.5 here was pinned to a version that fails 3 test
# files' collection on a 204-response-body assertion FastAPI added later).
#
# Install: pip install -r requirements.txt
#
# Most LLM/channel/voice provider SDKs below are lazily imported inside
# neuralcleave (try/except ImportError at the call site, matching the
# ModelRouter pattern) — the app runs fine without ALL of them installed,
# you only need the ones for the channels/providers you actually enable.
# They're all listed here anyway per the "one file, everything in it"
# policy; pip will install all of them.
